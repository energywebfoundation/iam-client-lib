IAM
====

The top-level class used to provide role management
 
..  autoclass:: IAM
    :members:
    
After instantiating of IAM you need to establish connection with decentralized 
identity registry (DID) and Ethereum Name with :function:`IAM#initializeConnection`.
To check connection status use :function:`IAM#isConnected`.
